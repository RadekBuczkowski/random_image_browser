Folders

In the Options dialog on the "Folders" tab you can set three folders to scan for images. The folder path can optionally be preceded with the "system:" keywoard to specify common system-defined folders. E.g. "system: MyPictures\Folder1" will load images from the user folder "c:\Users\[username]\Pictures\Folder1".

Folder Description
system: MyPictures The My Pictures folder
system: CommonPictures The file system directory that serves as a repository for image files common to all users
system: Desktop The user Desktop
system: Programs The directory that contains the user's program groups
system: MyDocuments The My Documents folder
system: ApplicationData The directory that serves as a common repository for application-specific data for the current roaming user.
system: LocalApplicationData The directory that serves as a common repository for application-specific data that is used by the current, non-roaming user.
system: InternetCache The directory that serves as a common repository for temporary Internet files
system: CommonApplicationData The directory that serves as a common repository for application-specific data that is used by all users.
system: Windows The Windows directory or SYSROOT
system: ProgramFiles The program files directory
system: ProgramFilesX86 The x86 Program Files folder

Options

The user.config file contains the configuration of the program. Most of the settings can be changed in the Options dialog. To open the file press F11 after closing this dialog.

Setting Description
ImageFolder1 First image folder (see below)
ImageFolder2 Second image folder (see below)
ImageFolder3 Third image folder (see below)
ImageExtensions Extensions of loaded images, comma-separated. Default: jpg, jpeg, gif, png, webp, avif
Layout Arrangement and number of images visible on screen. A value between 2 and 12. Default: 7
AutoScaling Auto-scaling mode when not zoomed: 1 = fit to outside edges, 2 = fit to inside edges, 3 = thumbnails with rounded corners. Default: 3
AutoScalingWhenZoomed Auto-scaling mode when zoomed: 1 = fit to outside edges, 2 = fit to inside edges, 3 = original size. Default: 1
ExtraZoom Additional zoom applied when in the zoomed mode. A value between 0.1 and 10. Value 1 means no extra zoom, less than 1 means image shrinked, and more than 1 means image enlarged. Default: 1
IsBlackBackground True = black background, or False = white background. Default: False
IsFullScreen True = full screen mode, or False = normal mode. Default: False
EnableAnimations True = animations enabled, or False = animations disabled. Default: True
GroupImages True = if images are grouped, where only the groups are shuffled. False = all images are in random order. If False, the mouse wheel changes meaning and is used to show the neighbor of an image instead of scrolling all images. To scroll all images, use the mouse wheel in the corners of the window. Default: True
AutoRestartEveryNImages Number of images scrolled until an automatic restart of the application is triggered, e.g. 500. Can solve problems with animation fluidity. Default: 0
DesiredFrameRate Desired frame rate per second for animations. Reducing the rate can improve performance at the cost of animation smoothness. Default: 60.
LargeImagePixelDimension Threshold in pixels for both width and height of images classified as large images. Large images will use low quality bitmap scaling algorithm to improve performance. Default: 1920 (FHD)
MaxImagePixelDimension Images with width or height larger than this value will have bitmaps scaled down when loading to improve performance. Images can later be scaled up if necessary to show them in original size, so only the image quality is affected. Default: 3840 (4K)
Language Specifies the language of the application, e.g. "en-US" and "da-DK". If not specified the language is the same as in the operating system. Default: not specified.
